# Construlike - Incorporadora

Aplicativo para incorporadora apresentar imóveis.

## Cores do Projeto

- **Core**: #fbba02 (Amarelo)
- **Fundo**: #1f2326 (Escuro)

## Funcionalidades

- Tela inicial com vídeo de fundo
- Menu lateral direito com opções de contato (WhatsApp, Telefone, E-mail)
- Menu lateral esquerdo com abas de navegação:
  - Perspectiva
  - Planta
  - Implantação

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
